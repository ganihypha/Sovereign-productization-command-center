## 🔐 8. COMMIT LOG PRINCIPLE

Tiap commit git = tiap 1 unit pekerjaan selesai. Format:

```
<type>: <short description>

<optional body explaining WHY>

Evidence: <link/screenshot/url>
```

**Types:** `feat`, `fix`, `docs`, `refactor`, `chore`, `deploy`, `canon`.

**Example:**
```
canon: freeze price architecture 5-tier — final

Decision made 22 April 2026. No more pricing changes until Bulan 6
review. Tier 1-5 range locked as per DOC 01 section 4.

Evidence: docs/01-ROADMAP.md#price-architecture
```

## 🛡️ 9. ALTERNATIF: APAKAH BUTUH TIM?

**Pertanyaan founder:** *"Apakah kita butuh tim? Atau bisa solo pakai AI agent?"*

### Scenario Solo + AI (0-Bulan-3 Validation)
✅ **Bisa** jika:
- Goal hanya PoC validation (belum scale)
- Cash runway limited (<Rp 100 jt)
- Belum ada paid client
- Founder teknis capable (TRUE — Haidar is)

**Tools AI leverage:**
- **Claude/GPT** untuk coding co-pilot
- **Cursor / Copilot** untuk dev speed
- **v0.dev / Lovable** untuk landing prototype
- **Clay.com / Apollo.io** untuk outbound automation
- **Mercury.com / Xendit** untuk banking/payment
- **Notion AI** untuk docs

**Estimated solo capacity:** 1 feature ship/minggu, 5 leads/minggu outbound, 1 pilot close/bulan.

### Scenario Tim (Bulan 3+ Scaling)
**Trigger untuk hire:**
- [ ] MRR ≥ Rp 15 jt (cover 1 FTE)
- [ ] Waiting list ≥ 5 prospect serius
- [ ] Founder > 60 jam/minggu 2 minggu berturut
- [ ] Technical debt > 2 feature backlog
- [ ] Ada 1 fungsi yang 100% bisa di-delegate

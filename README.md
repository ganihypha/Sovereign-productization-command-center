## 📖 User Guide

1. **Buka** https://sovereign-cmd-center.pages.dev
2. **Navigate** via sidebar kiri (klik menu)
3. **v7 Strategic Docs:** Scroll sidebar paling bawah → "📚 v7 Strategic Docs" section → klik dokumen (Master Index · Roadmap · Journal · Implementation · Team · Estimate)
4. **Read Markdown source:** Semua 6 dokumen juga tersedia di `docs/00-MASTER-INDEX.md` sampai `docs/05-ESTIMATE.md` untuk diff/version-control
5. **Daily Log (v6):** Sidebar "v6 Enhancements" → Daily Command Log → isi WBR → Save
6. **Pipeline CRM (v6):** Tambah lead → move stage dengan ← → → ekspor CSV
7. **Health KPI (v6):** Refresh → lihat skor agregat + rekomendasi aksi
8. **Data tersimpan otomatis** di browser (localStorage) — tidak ada login/signup
